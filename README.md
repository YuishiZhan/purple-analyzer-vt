# 🟣 Purple Analyzer with VirusTotal Enrichment

**Анализатор логов с автоматической проверкой IP-адресов через VirusTotal API.**

Этот инструмент помогает аналитикам Purple Team быстро выявлять и обогащать информацию о подозрительных IP-адресах из логов.

---

## 🚀 Возможности
- **Парсинг логов** — извлекает IP-адреса из строк лога.
- **Обнаружение брутфорса** — находит IP с большим количеством неудачных попыток.
- **Обогащение через VirusTotal** — проверяет подозрительные IP по глобальной базе угроз.
- **Генерация JSON-отчёта** — структурированный вывод для дальнейшего анализа.

---

      }
